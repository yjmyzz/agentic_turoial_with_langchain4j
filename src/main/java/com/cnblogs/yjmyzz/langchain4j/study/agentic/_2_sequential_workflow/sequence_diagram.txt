# 顺序工作流时序图

```mermaid
sequenceDiagram
    participant Main as 主程序(_2a_Sequential_Agent_Example)
    participant CvGenerator as CvGenerator
    participant CvTailor as CvTailor
    participant Model as ChatModel
    
    Main->>CvGenerator: 创建CvGenerator实例
    Main->>CvTailor: 创建CvTailor实例
    Main->>Main: 构建顺序工作流(tailoredCvGenerator)
    Main->>Main: 准备参数(lifeStory, instructions)
    Main->>CvGenerator: 调用generateCv(lifeStory)
    CvGenerator->>Model: 请求生成主简历
    Model-->>CvGenerator: 返回主简历(masterCv)
    CvGenerator-->>Main: 返回主简历(masterCv)
    Main->>CvTailor: 调用tailorCv(masterCv, instructions)
    CvTailor->>Model: 请求根据职位描述优化简历
    Model-->>CvTailor: 返回定制简历(tailoredCv)
    CvTailor-->>Main: 返回定制简历(tailoredCv)
    Main->>Main: 打印最终简历
```

## 交互说明

1. 主程序首先创建CvGenerator和CvTailor两个Agent实例
2. 使用sequenceBuilder将两个Agent连接成顺序工作流
3. 顺序工作流首先调用CvGenerator生成主简历
4. 然后将生成的主简历作为输入传递给CvTailor进行定制
5. 最终返回经过职位描述优化后的定制简历
